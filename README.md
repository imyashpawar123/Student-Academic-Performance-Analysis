# 📊 Student Academic Performance Analysis

A complete **Exploratory Data Analysis (EDA)** project using **Python, Pandas, NumPy, Matplotlib, and Seaborn** to analyze student academic performance data.

This project focuses on analyzing student scores in:

- 📘 Math
- 📗 Reading
- 📝 Writing

The analysis helps identify:

- Performance trends
- Gender-based score comparison
- Effect of test preparation courses
- Correlation between subjects
- Overall student performance

---

# 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

# 📂 Dataset Information

Dataset Used: **StudentsPerformance.csv**

The dataset contains student-related information including:

- Gender
- Race/Ethnicity
- Parental Level of Education
- Lunch Type
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score
